# **App Name**: Suraksha Kavach

## Core Features:

- User Registration & Verification: Secure registration using OTP verification with mobile number and Aadhaar (last 4 digits + verification ID only).
- Emergency Nominee Pre-registration: Allows insurance holders to pre-register an emergency nominee for consent in critical situations.
- Trust Score System: Dynamic trust score (0-100) based on premium payments, policy age, claim history, and fraud flags.
- Hospital Search & Verification: Hospitals can search for patients using mobile number and last 4 digits of Aadhaar after OTP verification. Doctors can also input diagnoses and upload timestamped immutable records.
- Automated Claim Approval Engine: Automatically approves claims when holder & hospital trust scores are high and the claim amount is within the safety cap. Claims failing the trust check get marked for conditional or manual review.
- AI-powered Anomaly Detection: Automatically detects fraudulent bills (duplicate items, inflated costs).
- AI Claim Explanations tool: Generate automated claim approval/rejection reasons and explanations: "Approved because…" or "On hold because…"

## Style Guidelines:

- Primary color: Forest green (#386641) to represent health and trust.
- Background color: Light beige (#F5F5DC), offering a clean and accessible backdrop.
- Accent color: Muted gold (#A38329) for highlights and calls to action.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif for clarity and readability.
- Simple and clear icons for policy status, claim timelines, and approval/rejection reasons.
- Clean and modular layout with a focus on accessibility and user-friendliness for all user roles.
- Subtle transitions and animations to guide users through the claim process.
